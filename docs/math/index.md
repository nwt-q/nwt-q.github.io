### 📐 第一部分：高等数学（微积分）

#### 一、函数、极限、连续（基础核心）

- 函数：定义域、值域、奇偶性、周期性、单调性、有界性

- 极限：定义（数列 / 函数）、性质（唯一性 / 有界性 / 保号性）、计算（等价无穷小替换 / 洛必达法则 / 泰勒公式 / 夹逼准则 / 单调有界准则）

- 连续：定义、间断点分类（第一类 / 第二类）、闭区间上连续函数的性质（有界性 / 最值定理 / 介值定理 / 零点定理）

- 🌟 重难点：泰勒公式的灵活运用、数列极限的证明、间断点判定

#### 二、一元函数微分学

- 导数：定义（左导 / 右导）、几何意义、基本求导公式、复合函数 / 隐函数 / 参数方程求导、高阶导数

- 微分：定义、几何意义、微分中值定理（罗尔定理 / 拉格朗日中值定理 / 柯西中值定理 / 泰勒中值定理）

- 应用：单调性与极值、凹凸性与拐点、曲率（数一 / 数二）、经济应用（数三：边际成本 / 收益 / 利润）

- 🌟 重难点：中值定理的证明题、不等式证明、极值与最值综合题

#### 三、一元函数积分学

- 不定积分：定义、基本积分公式、换元积分法（第一 / 第二类）、分部积分法、有理函数积分

- 定积分：定义（黎曼积分）、性质、微积分基本公式（牛顿 - 莱布尼茨公式）、反常积分（收敛性判定）

- 应用：面积、体积、弧长（数一 / 数二）、旋转体表面积（数一 / 数二）、物理应用（数一 / 数二：功 / 引力 / 压力）、经济应用（数三：消费者剩余 / 生产者剩余）

- 🌟 重难点：反常积分收敛性判定、积分不等式证明、综合应用计算题

#### 四、多元函数微分学

- 基础：二元函数定义域、极限、连续性、偏导数、全微分、高阶偏导数（混合偏导连续性）

- 应用：复合函数偏导数（链式法则）、隐函数求导（方程组形式）、方向导数与梯度（数一）、多元函数极值（无条件 / 条件极值拉格朗日乘数法）、曲面的切平面与法线（数一）

- 🌟 重难点：抽象复合函数偏导数、条件极值综合题

#### 五、多元函数积分学

- 二重积分：定义、性质、计算（直角坐标 / 极坐标）、对称性应用

- 三重积分：定义、计算（直角坐标 / 柱坐标 / 球坐标）、对称性（数一 / 数二）

- 曲线积分：第一类（对弧长）、第二类（对坐标）、格林公式、斯托克斯公式（数一）

- 曲面积分：第一类（对面积）、第二类（对坐标）、高斯公式（数一）

- 🌟 重难点：重积分对称性应用、线面积分与格林 / 高斯公式结合题

#### 六、无穷级数（数一 / 数三）

- 常数项级数：收敛 / 发散定义、性质、正项级数审敛法（比较 / 比值 / 根值）、交错级数（莱布尼茨判别法）、绝对收敛与条件收敛

- 幂级数：收敛半径 / 收敛域、和函数、泰勒级数展开

- 傅里叶级数（数一）：周期函数展开、正弦 / 余弦级数

- 🌟 重难点：幂级数和函数求解、级数收敛性判定

#### 七、常微分方程与差分方程

- 常微分方程：一阶（可分离变量 / 齐次 / 线性）、二阶线性齐次 / 非齐次方程（特征方程法 / 待定系数法）、欧拉方程（数一）

- 差分方程（数三）：一阶线性差分方程、通解与特解

- 应用：物理模型（数一 / 数二）、经济模型（数三）

- 🌟 重难点：二阶非齐次线性方程特解求解、微分方程应用题

### 🔢 第二部分：线性代数（全卷通用）

#### 一、行列式

- 定义：n 阶行列式的展开式、特殊行列式（对角 / 上三角 / 下三角 / 范德蒙行列式）

- 性质：转置不变、行 / 列变换、展开定理、克莱姆法则

- 🌟 重难点：高阶行列式计算、克莱姆法则应用

#### 二、矩阵

- 基础：矩阵定义、运算（加减 / 乘法 / 数乘 / 转置）、逆矩阵（定义 / 存在条件 / 求法）、伴随矩阵

- 进阶：矩阵的秩（定义 / 性质 / 求法）、初等矩阵（与初等变换的关系）、分块矩阵（运算 / 逆矩阵）

- 🌟 重难点：逆矩阵求解、矩阵秩的判定

#### 三、向量

- 线性相关性：线性表示、线性相关 / 无关的判定（定义 / 秩法）

- 向量组的秩：极大线性无关组、等价向量组

- 向量空间（数一）：基、维数、坐标变换、过渡矩阵

- 🌟 重难点：线性相关性判定、极大无关组求解

#### 四、线性方程组

- 齐次方程组：解的结构（基础解系）、通解形式

- 非齐次方程组：解的存在性（判定定理）、解的结构（齐次通解 + 非齐次特解）

- 🌟 重难点：含参数方程组的求解、基础解系构造

#### 五、特征值与特征向量

- 定义：特征值 / 特征向量的求解、性质（迹 / 行列式关系）

- 相似对角化：相似矩阵的性质、对角化的充要条件、实对称矩阵的对角化（正交矩阵）

- 🌟 重难点：实对称矩阵对角化、特征值与特征向量综合题

#### 六、二次型

- 定义：二次型的矩阵表示、秩

- 标准化：配方法、正交变换法（合同变换）

- 正定性：判定（顺序主子式 / 特征值法）

- 🌟 重难点：正交变换化二次型为标准形、正定性判定

### 📊 第三部分：概率论与数理统计（数一 / 数三）

#### 一、随机事件与概率

- 事件：关系（包含 / 互斥 / 对立）、运算（和 / 积 / 差）、样本空间

- 概率：定义、性质、古典概型 / 几何概型、条件概率、乘法公式、全概率公式、贝叶斯公式

- 独立性：事件独立的定义、伯努利概型

- 🌟 重难点：全概率公式与贝叶斯公式应用

#### 二、随机变量及其分布

- 离散型：分布律（二项 / 泊松 / 超几何分布）、分布函数

- 连续型：概率密度（均匀 / 指数 / 正态分布）、分布函数

- 函数的分布：离散型 / 连续型随机变量函数的分布

- 🌟 重难点：连续型随机变量函数的分布求解

#### 三、多维随机变量及其分布

- 联合分布：离散型（联合分布律）、连续型（联合概率密度）、联合分布函数

- 边缘分布与条件分布：边缘分布律 / 密度、条件分布律 / 密度

- 独立性：随机变量独立的判定

- 函数的分布：和 / 积 / 商分布、最大值 / 最小值分布

- 🌟 重难点：多维随机变量函数的分布、条件概率计算

#### 四、数字特征

- 核心：期望、方差、协方差、相关系数

- 性质：期望 / 方差的运算性质、协方差与相关系数的关系、不相关性与独立性

- 常见分布的数字特征：二项 / 泊松 / 均匀 / 指数 / 正态分布

- 🌟 重难点：协方差与相关系数计算、抽象随机变量数字特征

#### 五、大数定律与中心极限定理

- 大数定律：切比雪夫 / 辛钦大数定律

- 中心极限定理：列维 - 林德伯格（独立同分布）、棣莫弗 - 拉普拉斯（二项分布逼近正态）

- 🌟 重难点：中心极限定理的应用（概率近似计算）

#### 六、数理统计的基本概念（数一 / 数三）

- 样本：简单随机样本、样本均值 / 样本方差 / 样本矩

- 常用统计量分布：χ² 分布、t 分布、F 分布（性质 / 分位数）

- 抽样分布：单正态总体样本均值 / 方差的分布、双正态总体统计量分布

- 🌟 重难点：统计量分布的判定

#### 七、参数估计

- 点估计：矩估计法、最大似然估计法

- 区间估计（数一）：单正态总体均值 / 方差的置信区间、双正态总体参数置信区间

- 估计量的评价标准：无偏性、有效性、一致性

- 🌟 重难点：最大似然估计、矩估计的计算

#### 八、假设检验（数一）

- 基本概念：原假设 / 备择假设、显著性水平、拒绝域、两类错误

- 单正态总体：均值 / 方差的假设检验（Z 检验 /t 检验 /χ² 检验）

- 双正态总体：均值差 / 方差比的假设检验（Z 检验 /t 检验 / F 检验）

- 🌟 重难点：假设检验的拒绝域求解
