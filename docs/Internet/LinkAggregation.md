# 链路聚合典型配置



## 配置手工模式链路聚合

以太网链路聚合是指将多条以太网物理链路捆绑在一起成为一条逻辑链路，从而实现 增加链路带宽的目的。链路聚合分为手工模式和LACP模式。

手工模式下，Eth-Trunk的建立、成员接口的加入由手工配置，没有链路聚合控制协议 LACP的参与。当需要在两个直连设备间提供一个较大的链路带宽而设备又不支持LACP 协议时，可以使用手工模式。手工模式可以实现增加带宽、提高可靠性、负载分担的 目的。

手工模式下，所有的活动链路都参与数据转发并分担流量。



**配置注意事项**

- 一个Eth-Trunk接口中的成员接口必须是以太网类型和速率相同的接口。
-  Eth-Trunk链路两端相连的物理接口的数量、速率、双工方式、流控配置必须一 致。
- 如果本端设备接口加入了Eth-Trunk，与该接口直连的对端接口也必须加入Eth Trunk，两端才能正常通信。
- 两台设备对接时需要保证两端设备上链路聚合的模式一致。
- 本例适用于S600-E交换机的所有版本所有产品。
