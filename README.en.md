## 🌟 Project Overview

**ELNB** (Endless Learning & Notes Base) is a comprehensive technical knowledge base platform built with VitePress, designed to provide frontend developers with a systematic and structured knowledge sharing and storage space.

This project is not just a simple documentation site, but a complete knowledge management system covering a wide range of content from basic frontend technologies to advanced backend development.

### ✨ Core Features

- 📝 **Markdown-Driven** - Quickly create and organize content using familiar Markdown syntax
- ⚡ **Lightning Fast** - Instant server startup, lightning-fast hot updates, and leverage Vite ecosystem plugins
- 🎨 **Modern Theme** - Beautiful default theme and customizable UI components
- 🌐 **Multi-language Support** - Support for multi-language documentation (Chinese, English, Japanese, Korean, etc.)
- 📊 **Math Formula Support** - Built-in KaTeX support for perfect mathematical formula rendering
- 🔄 **Flowchart Support** - Integrated Mermaid for easy professional flowchart creation
- 🎯 **Component-Based** - Support for Vue component embedding for high customization

## 📖 Quick Start

### Prerequisites

- Node.js 18+ 
- pnpm 8.6.10+

### Installation

```bash
# Clone the repository
git clone https://github.com/nwt-q/nwt-q.github.io.git
cd nwt-q.github.io

# Install dependencies
pnpm install

# Start development server
pnpm docs:dev
```

Visit `http://localhost:5173` to preview the site.

### Build for Production

```bash
# Build the site
pnpm docs:build

# Preview the build
pnpm docs:preview
```
